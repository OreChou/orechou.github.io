七日书 Vol.1

OreChou
2025-11-09
weekly

七日书。我即将开启的周刊系列的刊名，言简意赅的名字。
我很喜欢写文字这件事情，虽然我并不擅长。但我觉得一个人，如果能写自己喜欢的文字，输出想要输出的内容，是一件很开心的事情。所以我要成为一个开心的人。
我写的文字并不好阅读，但所有的文字均会从我指尖敲出，产生于我这颗人脑。可放心不会是大语言模型生成。

洞见 - Insight

让Agent系统更聪明之前，先让它能被信任

来自阿里云开发者微信公众号的文章，描述了 Agent 系统开发过程中，涉及到的工程复杂度问题：包含可运行性、可复现性和可进化性。其中可运行性由 Agent 框架帮开发人员解决，当我们通过几行代码调用 LLM 就能跑起来一个 Agent 后，我们可能会产生 Agent 系统很容易开发的错觉。但其实后两者才是我们要去面对和解决的工程问题。现在 Agent 的研发的关注点也从 Prompt engineering、Context engineering 演变到架构设计上。

AIOps 通用 Agent 探索

七猫对 AIOps 通用 Agent 的探索。使用了 ReAct 设计思想：即引导模型将复杂问题进行拆分，分布进行推理和行动，同时还引入了观察环节。在每次执行之后，都会先观察当前现状，然后再进行下一步的推理。该实践公开的仓库地址，在 Agent 的实现上使用了字节开源的 Eino 框架。另外本周还看了下阿里的这篇文章ReAct范式深度解析：从理论到LangGraph实践，也是对 ReAct 设计思想探讨实践。

Agentic Design Pattern

谷歌新书，关于智能体设计模式的，涵盖了智能体设计的核心概念和实践方法。如果英语看得比较费力，也有很多翻译的版本可以选择。其中一个中文翻译的 Github 项目。

器物 - Artifacts

ChatGTP-Atlas

浏览器作为当前主要的 AI 应用入口，厂商们也在陆续推出自己的浏览器应用。Atlas 是 ChatGTP 推出的自己的浏览器应用。之前开放给 Plus 用户，本周发现免费用户也可以下载使用了。现在切换浏览器很容易，初始化的时候可以选择导入电脑上的 Chrome 或者 Safari 的书签等数据，切换后并没有使用成本。不过免费用户使用 ChatGPT 的功能是有限的，模型会限制以及降级。

Comet

Comet 是 Perplexity 推出的浏览器应用，注册并使用即送一个月的 Pro 订阅。Comet 有一个比较好的任务功能，可以设置一些自动化的任务来执行并获得结果。

PS：关于 macOS 软件安装与更新

我一直有软件更新的强迫症（如果更新就想要立刻更新），然后对于系统也有一些洁癖（不喜欢把系统搞乱，例如垃圾文件、命名混乱等）。于是对于 macOS 系统的软件安装，我定了一个规则，只使用下面的三个安装版本：

  1. Web 版（浏览器就直接使用了，不安装软件）
  2. App Store 安装
  3. Homebrew 安装
    除此之外来源的应用不安装，除非特别需要或者一定需要（比如做调研之类）。另外 brew 可以每天执行命令更新，App Store 版的软件自动更新有延迟，所以使用 mas 命令来强制安装。搭配一下命令就是：

这样的好处是，每次打开命令行，执行一下 appupgrade 指令即可更新所有软件了。

游戏 - Game

部落冲突（Clash of clans）

上个月国庆假期间我又开始玩起了部落冲突。之前玩是在大学本科第四年的时候，那是 16 年吧，时隔了 9 年。玩这个的原因很简单，就想找一个方便但又不需要重度参与的游戏玩。但是 PS 、Switch 和 PC 都是会需要占据大量的时间，以及需要固定场地的。那么一个快速、随时、随地可玩，游戏性又比较强，然后又不怎么花钱，又兼顾对抗、养成、建设、经营，游戏 UI 又比较不错的游戏呢？于是我就想到了部落冲突。当初玩的时候大本营最高只有 10 级，现在游戏已经最高 17 级了，很多新的东西。另外我是用美区 AppleID 安装的国际版，因为登陆了之前玩的国服账号，里面太多跟游戏无关的活动内容了，就跟狗皮膏药一样。其实回归之后发现这个游戏也会占据一些注意力，因为会关注于建筑工人的建筑进度、科技研究进度等。

声栖 - Resonance

为你盛开-许巍《无尽光芒》巡回演唱会现场纪念

偶然听到了《只有爱》，然后就在 Apple Music 上找到整个 Live 专辑，上下班在车上一直听。这场 Live 里面《远航》、《时光》等都很不错。许巍的音乐很有故事性，很喜欢这种民谣加上摇滚的音乐风格。我一直憧憬去一些热门自驾路线上跑跑，放着自己构建的歌单（包含了许巍、毛不易、周杰伦等歌手），希望明年能够去实施。24 年底的时候 B 站上关注了一个叫”步履不停的后天”，是一个自行车骑行博主，拍摄了他自己从上海长江入海口出发一路沿着长江到达长江源头三江源的故事。非常佩服这种远航的毅力。

行间 - Between Days

这周老婆跟娃在龙头寺公园附近住，周一、周四上午回家，因为娃下午要在家附近上课。周四晚上下班回家后，发现娃用剪刀把猫的胡须剪掉了，看着猫”一脸幽怨”的样子，觉得十分好笑。
