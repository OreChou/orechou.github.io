macOS 软件

最近更新时间：2025.10.12

AI

  • ChatGPT
    OpenAI’s official ChatGPT desktop app。Free。
    安装理由：在 mac 系统而不是浏览器快速唤起 ChatGPT 客户端，提高使用效率。
    安装方式：brew。

  • Claude
    Anthropic’s official Claude AI desktop app。Free。
    安装理由：在 mac 系统而不是浏览器快速唤起 Claude 客户端，提高使用效率。
    安装方式：brew。

系统增强

  • Command X
    在 Finder 中通过 command x 剪切和 command v 粘贴文件。免费。
    安装方式：AppStore。

  • Reminders Menubar
    在 Menubar 中快捷操作 Reminders。免费。
    安装方式：brew。

  • PrettyClean
    好用的 macOS 磁盘清理工具。免费。
    安装方式：brew。

  • Keka
    压缩解压工具。免费。
    安装方式：brew。

  • raycast
    安装方式：brew。

  • Google Chrome
    谷歌浏览器。免费。
    安装方式：brew。

开发工具

  • iTerm2
    好用的 macOS 终端命令行软件。免费。
    安装方式：brew。

  • SnippetsLab
    代码片段管理软件。收费，但可以等限免时购买。
    安装方式：AppStore。

  • postman
    API 文档和调试工具。功能免费，高级服务收费。
    安装理由：缺点是比较重，优点是功能全面。
    安装方式：brew。

阅读

  • NetNewsWire
    免费的 RSS 订阅软件。免费。
    安装方式：AppStore。

  • calibre
    电子书管理软件。免费。
    安装方式：brew。
