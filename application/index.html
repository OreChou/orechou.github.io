macOS 软件

系统增强

  • Command X
    在 Finder 中通过 command x 剪切和 command v 粘贴文件。免费。
    安装方式：AppStore。

  • Reminders Menubar
    在 Menubar 中快捷操作 Reminders。免费。
    安装方式：brew。

  • PrettyClean
    好用的 macOS 磁盘清理工具。免费。
    安装方式：brew。

  • Keka
    压缩解压工具。免费。
    安装方式：brew。
